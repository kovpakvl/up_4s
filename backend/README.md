# SecureOffice backend

Новый серверный слой для схемы "админское desktop-ПО + web-кабинет сотрудника".

## Локальный запуск через Docker Compose

```powershell
docker compose up -d --build
```

После запуска API доступен по адресу:

```text
http://127.0.0.1:8765
```

Первичная проверка:

```powershell
curl http://127.0.0.1:8765/health
curl http://127.0.0.1:8765/api/status
```

## Веб-кабинет сотрудника

- `http://127.0.0.1:8765/activate` — активация по ключу сотрудника;
- `http://127.0.0.1:8765/login` — вход сотрудника;
- `http://127.0.0.1:8765/cabinet` — список личных парольных записей.

Ключи активации создаются через admin API. В следующей итерации эти действия
будут вынесены в desktop-интерфейс администратора.
